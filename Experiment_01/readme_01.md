# Experiment 01: Implementation of Basic Logic Circuits using Logisim

##  Objective
To understand and implement basic digital logic circuits using Logisim Evolution. This experiment focuses on designing and simulating fundamental components such as multiplexers, priority encoders, and foundational combinational logic circuits.

##  Background Study
Digital logic circuits form the backbone of computer architecture and are used to perform logical operations on binary inputs. Basic components such as AND, OR, and NOT gates are combined to form more complex circuits:

*   **Multiplexer (MUX):** A combinational circuit that selects one input from multiple inputs based on select lines and forwards it to the output.
*   **Priority Encoder:** A circuit used to encode multiple inputs into a binary format, giving priority to the highest-order input.
*   **Logisim Evolution:** A digital circuit simulation tool that allows users to design, test, and analyze circuits visually. It provides a user-friendly interface where components can be connected, and their behavior can be observed in real-time.

##  Experiment Description
In this experiment, various digital circuits were designed and simulated from first principles. The specific implementations include:

*   **8:1 Multiplexer:** Implemented to select inputs based on control signals. A hierarchical multiplexer design was also created to handle multiple input lines efficiently.
*   **8:3 Priority Encoder:** Designed to convert multiple input signals into a coded 3-bit output.
*   **Combinational Logic:** Circuits utilizing OR gates were implemented to understand signal flow and logical operations.

The outputs were verified by changing input values and observing the corresponding results.

##  Observations & Result
*   The simulation results showed that the **multiplexers** correctly selected inputs based on the select lines.
*   The **priority encoder** produced the expected binary output corresponding to the highest priority input.
*   The **combinational logic circuits** responded accurately to different input combinations, confirming the correctness of the design.
*   **Final Result:** The experiment was successfully completed, and the designed circuits were verified using Logisim. The behavior of these components was clearly understood through simulation, helping develop a strong understanding of practical combinational circuit implementation.

## 🎯 Conclusion
This experiment provided practical exposure to designing and simulating digital circuits using Logisim. It reinforced the theoretical concepts of logic gates, multiplexers, and encoders, and demonstrated how complex systems can be built using simple components. The use of simulation tools made it easier to visualize and verify circuit behavior effectively.
